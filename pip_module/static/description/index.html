Install Python Pip Packages

You can now download any pip modules in odoo.

Tested Packages:

  • pip install pandas
  • pip install matplotlib
  • pip install seaborn
  • pip install keras
  • pip install numpy
  • pip install pillow
  • pip install scipy
  • pip install requests

Steps To Use App

Step 1.

Go To Settings >> Scroll Down >> Active Developer Mode

Step 2.

Go To Settings >> Click On Tecnical From Menu >> Click On Pip Install

Step 3.

Write pip Cammand >> Click Execute >> Wait Untill Finish

Step 4.

Cammand prompt output will be show on user error

Thank You Enjoy Python Pip Packages :)
